// Stack Data Structure
Back / Forward Stack LIFO
A Stack follows Last In, First Out order. The browser maintains two stacks — one for back history and one for forward history. Clicking a link pushes the current page onto the back stack and clears the forward stack.
push() Visit new page → push to back stack
pop() Click Back → pop from back stack
peek() Check which page is on top
⏱ Time: O(1) push/pop  |  Space: O(n)
// Queue Data Structure
Tab Queue FIFO
A Queue follows First In, First Out order. Pages are added to the back of the queue (enqueue) and removed from the front (dequeue). This simulates a "tabs to open" list — like a playlist of pages.
enqueue() Add a page to the back of the queue
dequeue() Open the front page and remove it
front() Peek at the next page to open
⏱ Time: O(1) enqueue/dequeue  |  Space: O(n)
